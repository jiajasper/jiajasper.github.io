    myWizard is an interactive, real-time data visualisation tool to automate and monitor supply chain IT operation.



    The selected work is a collection of 10+ interactive live data visualization dashboards from the myWizards AIOps package. The interactive dashboards improve the transparency and efficiency of the supply chain IT operations at Coles Supermarket Australia. It is made with Splunk Processing Language and custom frontend codes. Please keep in mind that I am not allowed to show this project in greater details, the graphic materials are mostly renders that I made after I rolled off the project.
